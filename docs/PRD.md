# Freedive Sessions

## Vision
The best Apple Watch companion for recreational freediving and snorkeling.

## Product Positioning
Freedive Sessions is a session logger and memory platform, not a dive computer.

## Target Users
- Recreational freedivers
- Snorkelers
- Travelers
- Apple Watch Ultra owners

## Core User Stories
1. Start a session before entering water.
2. See current depth and dive duration underwater.
3. Track surface intervals.
4. Review dives after the session.
5. Share activities to Strava.
6. Build a personal dive history.

## MVP Features
- Watch-first session tracking
- Depth logging
- Dive detection
- Surface intervals
- Event markers
- iPhone charts
- Map view
- Strava export

## Future Features
- Heatmaps
- Photo attachments
- GoPro integration
- Dive classification
- Social sharing

## Non Goals
- Dive planning
- Decompression calculations
- Safety recommendations
- Professional training

## Success Metrics
- >80% dive detection accuracy
- <5% failed syncs
- Session completion rate >90%
