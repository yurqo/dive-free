# dive-free

The best Apple Watch companion for recreational freediving and snorkeling — a
session logger and memory platform (not a dive computer). See [`docs/`](docs/)
for the architecture notes, CI/CD runbook, and privacy policy.

## Project layout

The project is **modular** and the Xcode workspace is **generated by [Tuist](https://tuist.dev)**
from [`Project.swift`](Project.swift) — so the `.xcodeproj`/`.xcworkspace` are not
committed (they're regenerated on demand).

```
Project.swift            # Tuist manifest: all targets + dependencies
Apps/
  WatchApp/              # DiveFreeWatch — watchOS app (session tracking)
  iPhoneApp/             # DiveFree — iOS app (history, charts, Strava)
Packages/
  Domain/                # Pure models + DiveDetector (no dependencies)
  Persistence/           # SwiftData models & store
  Sensors/               # DepthProvider + SensorManager (Watch Ultra depth)
  Sync/                  # WatchConnectivity SyncManager
  Strava/                # Strava export client
```

Dependency direction: apps → feature packages → `Domain`. `Domain` depends on
nothing, which keeps the business logic (e.g. dive detection) fast to test.

## Requirements

- **Xcode 16+** (full Xcode, not just Command Line Tools) — required to build the
  watchOS/iOS apps and anything using HealthKit/CoreMotion/WatchConnectivity.
- **Tuist** — `brew install tuist`
- Swift 6 toolchain (bundled with Xcode)

## Getting started

```sh
# 1. Generate the Xcode workspace from the manifest
tuist generate

# 2. Open it
open DiveFree.xcworkspace
```

Pick the **DiveFree** scheme to run the iPhone app, or **DiveFreeWatch** for the
watch app. After editing `Project.swift` (adding files/targets/dependencies), run
`tuist generate` again.

### Running tests

Each package has a unit-test target (`DomainTests`, `PersistenceTests`, etc.). Run
them from Xcode (⌘U) or:

```sh
tuist test
```

## Status

DiveFree ships on TestFlight and is preparing for App Store release. Implemented:
watch-first session capture with `CMWaterSubmersionManager` depth, automatic dive
detection, hands-free Digital Crown + Action-button control, HealthKit workouts,
watch↔phone sync, and an iPhone history with depth charts, dive-spot maps,
photo/video attachments, voice notes, and Strava export.
