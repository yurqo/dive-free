## Project overview

DiveFree is a SwiftUI freediving session logger for Apple Watch Ultra + iPhone. Records depth via `CMWaterSubmersionManager`, detects dives, saves sessions via SwiftData, syncs to iPhone via WatchConnectivity. Not a dive computer.

Swift 6, `SWIFT_STRICT_CONCURRENCY = complete`, `@Observable` + `@MainActor` throughout.

## Workspace setup

`.xcodeproj`/`.xcworkspace` are **not committed** — generated by Tuist from `Project.swift`. Regenerate after cloning or changing `Project.swift`:

```sh
tuist generate --no-open
```

Tuist version is pinned in `mise.toml`.

## Commands

```sh
tuist test                          # all package tests

# single package (replace scheme name as needed)
xcodebuild test -workspace DiveFree.xcworkspace -scheme Domain \
  -destination "platform=iOS Simulator,name=iPhone 17" CODE_SIGNING_ALLOWED=NO | xcbeautify

xcodebuild build -workspace DiveFree.xcworkspace -scheme DiveFree \
  -destination "generic/platform=iOS Simulator" \
  CODE_SIGNING_ALLOWED=NO EXCLUDED_ARCHS=x86_64 | xcbeautify
```

`EXCLUDED_ARCHS=x86_64` is required on Apple Silicon builds.

## Architecture

```
DiveFree (iPhone)  DiveFreeWatch (Watch)
        \                  /
    Strava  Persistence  Sensors  Sync
                  \       |      /
                        Domain
```

`Domain` has no dependencies — models + `DiveDetector` only. Packages do not depend on each other. `SessionCoordinator` (Watch app) is the glue layer: it owns `WorkoutController`, `SensorManager`, `DiveDetector`, and `SyncManager`.
